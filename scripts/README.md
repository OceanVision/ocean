/scripts
========

ocean_exemplary_data.py
-----------------------

With this script graph database will be erased and rss urls from `data/rss_feeds` (default) file will be added.

You can get `rss_feeds` file from Ocean big data server or generate it yourself with `/graph_workers/web_crawler_exporter.py` (run this script for details).

ocean_small_exemplary_data.py
-----------------------------

This script will erase whole graph database and will populate it with a few exemplary nodes.
